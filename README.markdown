# 코딩냥 노래봇

당신이 만든 최강의 한국어 디스코드 노래봇입니다! 😺 Discord.js v14, ytdl-core, yt-search, @discordjs/voice, ffmpeg-static을 사용해 유튜브 음악을 재생하며, 화려한 임베드와 풍부한 기능으로 최고의 음악 경험을 제공해요.

## 설치 방법
1. **Node.js 설치**: Node.js 16 이상을 설치하세요.
2. **의존성 설치**:
   ```bash
   npm install discord.js ytdl-core yt-search @discordjs/voice fluent-ffmpeg ffmpeg-static
   ```
3. **봇 설정**:
   - [Discord Developer Portal](https://discord.com/developers/applications)에서 봇을 만들고 토큰을 `config.json`의 `YOUR_BOT_TOKEN_HERE`에 붙여넣으세요.
   - 봇 초대 시 `applications.commands`, 음성 채널 연결/말하기 권한을 부여하세요.
4. **봇 실행**:
   ```bash
   node index.js
   ```

## 명령어
- `/재생 <URL 또는 검색어>`: 유튜브 URL 또는 검색어로 노래를 재생하거나 대기열에 추가.
- `/검색 <쿼리>`: 검색어로 최대 5개 노래를 찾아 드롭다운 메뉴로 선택.
- `/재생목록 <URL>`: 유튜브 재생목록의 모든 곡을 대기열에 추가.
- `/정지`: 음악 재생을 멈추고 대기열을 초기화.
- `/스킵`: 현재 노래를 건너뛰기.
- `/대기열`: 현재 대기열 목록 표시 (최대 10곡).
- `/삭제 <번호>`: 대기열에서 특정 곡 삭제.
- `/셔플`: 대기열을 무작위로 섞기.
- `/볼륨 <1-100>`: 재생 볼륨 조절.
- `/지금재생`: 현재 재생 중인 노래 정보와 진행바.
- `/일시정지`: 노래를 일시정지.
- `/재개`: 일시정지된 노래를 재개.
- `/이동 <초>`: 노래를 특정 시간(초)으로 이동.
- `/루프 <모드>`: 반복 모드 설정 (끄기, 한 곡, 대기열).
- `/이퀄라이저 <모드>`: 음질 모드 설정 (기본, 베이스 부스트, 팝).

## 기능
- **한국어 명령어**: 직관적이고 친근한 한국어 슬래시 명령어.
- **화려한 임베드**: #55FF55(초록), #FF5555(빨강), #FFAA00(주황) 색상, 고양이 이모지(😺😿😸), 썸네일, 타임스탬프, 추가 필드, 진행바로 시각적 완성도 UP!
- **유튜브 검색**: `/검색`으로 상위 5개 결과 중 드롭다운 메뉴로 선택.
- **재생목록 지원**: `/재생목록`으로 유튜브 재생목록 전체 추가.
- **대기열 관리**: 최대 10곡 표시, `/삭제`로 특정 곡 제거, `/셔플`로 무작위 재정렬.
- **진행바**: `/지금재생`에 현재 재생 시간과 진행바 표시.
- **볼륨 조절**: 1~100%로 자유로운 볼륨 설정.
- **시간 이동**: `/이동`으로 원하는 재생 위치로 이동.
- **일시정지/재개**: 재생 제어 완벽 지원.
- **반복 모드**: `/루프`로 한 곡 또는 대기열 반복.
- **이퀄라이저**: ffmpeg를 사용해 실제 오디오 필터 적용 (기본, 베이스 부스트, 팝).

## 주의사항
- 봇이 음성 채널에 연결/말하기 권한이 필요해요.
- `/검색`은 상위 5개 결과를 드롭다운 메뉴로 제공.
- ffmpeg-static이 설치되어 있어야 이퀄라이저 필터가 적용됩니다.
- 만든 사람: 당신! 😺 (코딩냥 노래봇은 당신의 작품입니다!)

## 추가 제안
- **멀티 소스 지원**: Spotify, SoundCloud 추가.
- **자동 재생**: 대기열이 끝나면 추천 곡 자동 추가.
- **봇 상태 표시**: 현재 재생 중인 곡을 봇 상태에 표시.
